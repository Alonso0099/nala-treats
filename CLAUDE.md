# Nala's Treats Repostería

Landing page for a real Costa Rican bakery business. Static site: vanilla HTML/CSS/JS only — no frameworks or CSS libraries (React, Vue, Next.js, Tailwind, Bootstrap, etc.) without asking the user first and explaining tradeoffs. The user is learning frontend dev and needs to be able to understand and maintain every part of this code, so keep explanations included when introducing anything non-trivial.

## Structure
- `index.html` — page structure/content
- `styles/styles.css` — all styling, CSS variables for colors/fonts at the top
- `scripts/main.js` — product/gallery data arrays + rendering, nav toggle, footer year, scroll reveal
- `images/` — currently placeholder SVGs; real photos will replace these later (see README)
